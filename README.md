# LASTPARADISEAssignmentUE5

전투 아이템 기획 프로토타입

Unreal Engine Version: 5.6.1 

Project Type: ThirdPerson Template 기반.


이펙트가 로드되는데 시간이 걸릴 수 있습니다.

시간이 지나면 자동으로 이펙트가 로드되어 재생됩니다.

그래도 이펙트가 나타나지 않는다면 ESC키를 통해 Restart버튼을 눌러주세요. 

(언리얼 엔진 에디터 플레이에서 ESC를 누른다면 에디터 플레이를 종료하는 버튼과 겹쳐서 Restart버튼이 나타나지 않고 바로 종료 될 수 있습니다. 에디터 설정 편집에서 플레이 종료 버튼을 다른 키로 바꾸고 실행하시기 바랍니다.)


조작키

WASD - 이동

Space Bar - 점프

ESC - 메뉴

C - 암살 중지

F - 아이템 장착/암살

G - 아이템 버리기

R - 수리

Q - 투척 공격

마우스 XY 축 이동 - 화면 회전

마우스 왼쪽 버튼 - 기본 공격 / 조준

마우스 오른쪽 버튼 - 방어 / 방어 해제
